# Implementation Steps

## Local
1. Unzip the package.
2. Open the folder in VS Code.
3. Run `npm install`.
4. Run `npm run test`.
5. Run `npm run dev`.

## GitHub
1. Create a new repository named `mason-forge-bootstrap`.
2. Copy these files into the repo.
3. Commit with message: `Initialize Mason Forge bootstrap`.
4. Push to GitHub.

## Vercel
1. Import the GitHub repo into Vercel.
2. Framework: Vite.
3. Build command: `npm run build`.
4. Output directory: `dist`.
5. Deploy after approval.

## Supabase / Cloudflare Later
Use the included config as the first integration map. Do not add secrets to GitHub. Use environment variables only.
